Intrucciones para patchear un proyecto

[
	{ "action": string, "value": string },
	{ "action": string, "value": string },
	{ "action": string, "value": string },
	...
]


Donde action puede tomar uno de los siguientes valores:
- ADD_MEMBER, value: nombre del usuario a agregar
- REMOVE_MEMEBER, value: nombre del usuario a quitar del proyecto
- RENAME, value: nuevo nombre para el proyecto
